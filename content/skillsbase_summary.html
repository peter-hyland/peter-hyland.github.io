Skillsbase Course Summary

course_name: Installation and Maintenance of Customer Service Point (CSP) Demarcation Box

overview: This training course covers the installation and maintenance procedures for the Customer Service Point (CSP) Demarcation Box used in telecommunications. It includes instructions on mounting the CSP, routing cables, splicing fibers, conducting site surveys, and ensuring safety measures.

topics_covered

    topics_covered

    topic: CSP Box Lid Features

    detail: The CSP box lid includes a unique locking mechanism at 45 degrees.

    topics_covered

    topic: Primary Function of CSP Demarcation Box

    detail: The primary function of the CSP Demarcation Box is for the installation of copper connections.

    topics_covered

    topic: Sealing CSP after Installation

    detail: The CSP automatically seals when the lid is closed.

    topics_covered

    topic: Mounting CSP to Customer's Wall

    detail: The CSP should be mounted using glue.

    topics_covered

    topic: List of Tools for Installation

    detail: A list of tools needed for installation is not specified, and any available tools can be used.

    topics_covered

    topic: Layer of Inside/Out Cable for Outside Use

    detail: Only the outer sheathing of the Inside/Out cable is UV protected for outdoor use.

    topics_covered

    topic: Equipment for Cable Bend Radius Management

    detail: Inside/Out Cable is used to manage the bend radius of the cable.

    topics_covered

    topic: Length Measurement for Cable Installation

    detail: Only the length from the CSP to the ONT needs to be taken into account.

    topics_covered

    topic: Storage of Outgoing Cable Slack

    detail: The outgoing Inside/Out cable slack should be stored in the inner slack storage of the CSP.

    topics_covered

    topic: Tool for Stripping Inside/Out Fibre Cable

    detail: An open-edged blade is used for stripping the Inside/Out Fibre Cable.
