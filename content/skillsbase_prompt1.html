How to Create an Account

Steps: Email info@skillsbase.io to have an account registered in your name. Receive login details after creation.

Important Note: Operator account credentials cannot be used for the SkillsBase App.

How to Sign In

Process: Open web browser and navigate to https://operator.skillsbase.io/. Enter login details received via email.

Changing your Password

Recommendation: Change password by accessing 'My Account' and selecting 'Change Password'.

Using the Dashboard

Features

  • Home Screen with unique statistics
  • Sidebar menu for navigation and options like User Management, Courses, Products, Resources, etc.

User Management

Functionalities

  • Viewing users with filter and export options
  • Creating Users with details like Name, Email, Group assignment, and Assessor status
  • Setting Users' Passwords manually

Course Management

Key Aspects

  • Overview of created courses
  • Managing Course Blocks and Modules
  • Creating Courses and Modules with descriptions, images, and custom settings

Submodule Templates

Details: Various templates available for submodule content like videos, quizzes, images, checklists, and more.

Troubleshooting

Common Issues

  • Sign-in problems, Password resets
  • Assistance for User login issues
  • Notification settings for Push Notifications
  • Team Member addition and User Email modification
