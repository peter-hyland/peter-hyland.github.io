How to Create an Account

Description: Details on how to request and create an account to access the Operator Dashboard.

Key Points

  • No automatic account creation process available
  • Need to contact info@skillsbase.io for registration
  • Separate account needed for the SkillsBase App

How to Sign In

Description: Guidelines on how to sign in to the Operator Dashboard once an account has been created.

Key Points

  • Navigate to https://operator.skillsbase.io/
  • Use login details received via email

Changing your Password

Description: Steps to change the password once logged in to the Dashboard.

Key Points

  • Change password under 'My Account'
  • Recommended to update password for security

Using the Dashboard

Description: Instructions on navigating and utilizing the Home Screen and Sidebar Menu features.

Key Points

  • Access to user, course, and product details
  • Options to create, manage users, and view reports

User Management

Description: Managing user accounts, creating users, setting passwords, and assigning courses.

Key Points

  • View, create, edit, and delete user accounts
  • Assign courses and monitor user progress

Course Management

Description: Handling course-related tasks like creating, editing, and managing course modules.

Key Points

  • Generate course links, create, and customize courses
  • Filter, view, and modify courses

Submodule Templates

Description: Explains various submodule types available to create interactive content within modules.

Key Points

  • Options like Title Slide, Video Player, Mini Quiz, and more
  • Customize module content presentation

Troubleshooting

Description: Common issues faced along with solutions to problems related to login, password, user management, etc.

Key Points

  • Resolving sign-in, password, user login issues
  • Guidance on adding team members and changing user email
