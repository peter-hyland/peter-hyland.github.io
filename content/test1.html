How to Create an Account

Description: To access the Operator Dashboard (Expert), an account is required. Account creation is not automatic; contact info@skillsbase.io for registration. Operator account credentials are distinct from SkillsBase App.

How to Sign In

Description: After account creation, sign in at https://operator.skillsbase.io/ using the provided login details.

Changing your Password

Description: Password can be changed via 'My Account' after logging in. Navigate to 'My Account' and select 'Change Password'.
