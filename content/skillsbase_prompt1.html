How to Create an Account

Summary: To sign into the Operator Dashboard, an account needs to be created by contacting info@skillsbase.io. Separate registration is required for the SkillsBase App.

Key Points

  • Email info@skillsbase.io to have an account created
  • Separate registration needed for SkillsBase App

How to Sign In

Summary: Open the web browser, go to https://operator.skillsbase.io/, and enter the login details received via email.

Key Points

  • Access Dashboard using URL provided
  • Enter login details received via email

Changing your Password

Summary: While on the Dashboard home screen, change the password by accessing 'My Account' and selecting 'Change Password'.

Key Points

  • Change password on Dashboard home screen
  • Access 'My Account' to change password

Using the Dashboard

Summary: The Dashboard home screen provides quick access to users, groups, courses, modules, products, and resources. Sidebar menu offers more options.

Key Points

  • Dashboard displays key statistics
  • Sidebar menu for additional features

User Management

Summary: View and manage users, create new user accounts, assign courses, and track training progress.

Key Points

  • View, create, and manage user accounts
  • Assign courses and track progress

Course Management

Key Points

Submodule Templates
