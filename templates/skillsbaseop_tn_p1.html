How to Create an Account

Description: To sign into the Operator Dashboard (Expert), an account must be created by contacting info@skillsbase.io. Login details will be provided after the account creation.

Key Steps

  • Contact info@skillsbase.io to create an account
  • Receive login details after account creation

How to Sign In

Description: Open a web browser and navigate to https://operator.skillsbase.io/. Enter the login details received via email.

Key Steps

  • Access https://operator.skillsbase.io/
  • Enter login details

Changing your Password

Description: Log in to the Dashboard, change password by clicking 'My Account' and then 'Change Password'.

Key Steps

  • Navigate to 'My Account'
  • Select 'Change Password'

Using the Dashboard

Description: Navigate the Dashboard offering options like Users, Groups, Courses, Modules, Products, Resources, Jobs, News, Reports, Team Management, and My Account.

Key Features

  • Access quick view options
  • Navigate through different sections

User Management

Description: View, create, manage users, assign courses, and track progress. Includes features like filtering, exporting data, and managing actions for individual users.

Key Features

  • View user information
  • Create new users
  • Assign courses
  • Track training progress

Course Management

Description: Overview of course creation, modules, viewing, editing, and filtering. Includes functionalities like creating, editing, publishing courses, setting prerequisites, and managing course blocks.

Key Features

  • Create and edit courses
  • Organize modules and submodules
  • Set prerequisites and completion criteria

Troubleshooting

Description: Address common issues like sign-in problems, forgotten passwords, user log-in issues, push notification failures, adding team members, changing user emails, and providing a contact point for unresolved issues.

Key Solutions

  • Double-check URL for sign-in issues
  • Reset password if forgotten
  • Resend sign-up info to users
  • Check notification settings for push notification issues
  • Add team members through 'Team Management'
  • Edit user emails via 'Manage Users'
  • Contact Mark Horgan for unlisted problems
