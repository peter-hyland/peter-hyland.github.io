How to Create an Account

Description: Guide on creating an account to access the Operator Dashboard

Steps

  • Send an email to info@skillsbase.io to register an account
  • Receive login details after the account is created
  • Operator account credentials cannot be used for the SkillsBase App

How to Sign In

Description: Instructions for signing in to the Operator Dashboard

Steps

  • Open web browser and navigate to https://operator.skillsbase.io/
  • Enter login details received via email

Changing your Password

Description: Guide to changing password after logging in to the Dashboard

Steps

  • Click 'My Account' to access profile settings
  • Navigate to 'Change Password' and input new password
