How to Create an Account

Description: Need to have an account to use the Operator Dashboard. Accounts are to be registered via email request.

Note: Operator account credentials are different from the SkillsBase App credentials.

How to Sign In

Instructions: Navigate to https://operator.skillsbase.io/ and enter login details sent via email.

Changing your Password

Steps: After logging in, access 'My Account' to change the password through the 'Change Password' option.

Using the Dashboard

Home Screen: Dashboard home page provides navigation and quick view options for Users, Groups, Courses, Modules, Products, and Resources.

Sidebar menu: Contains links for Dashboard home, Courses, Users, Products, Product Types, Resources, Jobs, News, Reports, Team Management, and My Account.

User Management

Viewing users: Allows viewing and managing all user accounts with options to filter, search, export, and import user data.

Creating Users: Users can be created manually with required fields Name and Email. Option available for assigning user groups and setting temporary passwords.

Setting a Usersâ€™ Password: Manual password setting available with specific criteria for password complexity.

Assigning Courses to a User: Courses can be assigned to users on the 'Edit user' page under the 'Assign Courses' Tab.

Viewing Users Training Course Progress: Available under the 'Action' menu for each user, providing details on user's course progress and allowing actions like resetting module progression.

Course Management

Courses Overview: A course is composed of Modules which in turn are made up of Submodule Templates. Features for creating, duplicating, and managing courses are outlined.

How to Create a Course: Details on how to create a new course including setting it as default, adding a title, description, primary image, keywords, and activating the course.

How to Create a Module: Guide for creating a module within a course, including options for notifications, adding a title, description, primary image, and activation status.

How to Create Submodules: Instructions on adding content to modules by creating submodules from various templates.

Progression Settings: Customizing user progression through unlocking criteria, selecting prerequisite submodules and setting completion criteria.

Submodule Templates

List of Submodule Types: Includes Title Slide, Web Viewer, Video Player, Image Gallery, Itemised List, Dos and Don'ts Scenarios, Mini Quiz, Step-by-Step Video Guide, and more.

Specific Templates: Detailed instructions on how to use each submodule template, including settings for title, media uploads, item description, scenario creation, question types, and document uploads.

Troubleshooting

Common Issues: Solutions provided for issues related to signing into the dashboard, forgotten passwords, login troubles, push notifications, adding team members, changing user emails, and unlisted problems.
