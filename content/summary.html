Course Summary

Overview

SkillsBase App provides a platform for Experts to manage operator dashboard and user training courses. It covers creating account, signing in, changing password, using the dashboard, user and course management, various submodule templates, and troubleshooting tips.

How to Create an Account

To sign into the Operator Dashboard, you must have an account created by contacting info@skillsbase.io. Account credentials for the Operator Dashboard are separate from the SkillsBase App account.

How to Sign In

After creating an account, go to https://operator.skillsbase.io/ and log in using the credentials received via email.

Changing your Password

Users can change their password by accessing the 'My Account' section and selecting 'Change Password'.

Using the Dashboard

The Dashboard provides quick access to user, group, course, module, product, and resource management options.

User Management

Enables viewing, creating, editing, and assigning courses to users. Also, provides features to manage user progress and accounts.

Course Management

Includes creating, viewing, and editing courses, modules, and submodules. Users can customize progression settings for courses.

Submodule Templates

Offers various templates like Title Slide, Web Viewer, Video Player, Image Gallery, Itemised List, Dos and Don'ts Scenarios, Mini Quiz, Step-by-Step Video Guide, Hotspot Image, Simple Question, Hotspot Uploader, Question Bank Quiz, Upload Media, Checklist, and Document Viewer.

Troubleshooting

Provides solutions for common issues like sign-in problems, forgotten passwords, user login issues, push notification failures, adding team members, changing a user's registered email, and contact information for further assistance.
