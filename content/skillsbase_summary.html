How to Create an Account

Summary: To sign into the Operator Dashboard, an account must be created by emailing info@skillsbase.io.

Action: No facility to create an account automatically.

How to Sign In

Steps: Open the web browser and navigate to operator.skillsbase.io, then enter the login details received via email.

Changing your Password

Instructions: Access the Dashboard home screen, recommend changing password by clicking 'My Account' or profile picture, then 'Change Password.'

Using the Dashboard

Features: Quick view options, users, groups, courses, modules, products, resources, and sidebar menu for navigation.

User Management

Tasks: Viewing users, creating users, setting a user's password, assigning courses, and viewing users training course progress.
