How to Create an Account

Description: To sign into the Operator Dashboard, an account needs to be created by contacting info@skillsbase.io. Account credentials are not shared with the SkillsBase App.

Key Steps

  • Email info@skillsbase.io to request an account creation
  • Receive login details via email

How to Sign In

Description: After creating an account, sign in using the link provided in the email using the login details received.

Changing your Password

Description: After logging in, change the password by accessing the 'My Account' section and selecting 'Change Password'. Follow the instructions to set a new password.

Using the Dashboard

Description: Dashboard provides quick view options for Users, Groups, Courses, Modules, Products, and Resources. Sidebar menu allows navigation to different sections for managing users and content.

User Management

Description: View, create, and manage user accounts. Assign courses, set passwords, and track training progress of users.

Course Management

Description: Create, edit, and manage training courses. Add modules and submodule templates such as quizzes, videos, and image galleries. Set custom progression settings.

Submodule Templates

Description: Features various submodule types such as Title Slide, Video Player, Image Gallery, Quiz, and Document Viewer. Each template has specific instructions for content creation.

Troubleshooting

Description: Provides solutions for common issues like login problems, forgotten passwords, user login troubles, push notification errors, adding team members, changing user email, and contacting support for unlisted problems.
